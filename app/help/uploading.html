Uploading Files into MicrobeTrace

The Home page contains several sections that allow you to select and view information, and consists of the following sections:

  • Preparing files for upload
  • File Types
  • Rules

View one indicator for all locations

Uploading

  • Double click on the link provided to you or type http://apps.nccd.cdc.gov/HIVTraceS into the address field in your browser.
    Result: The system displays the HIV Trace Select Files page.



  • The Submit button is not enabled until you choose a file.
  • Click the Choose file button.
    Result: The system displays a windows page for navigating to the desired file.

  • Navigate to the desired file and double-click on it.
    Result: The system selects the file and enables the Submit button.
  • Click Submit.
    Result: The system starts loading the file and displays a progress bar. You can click the Details link to view more information about the file and progress.


    Upon completing the upload, the system displays the nodes.
  • At the bottom right of the page, the system displays the number of nodes in the file, the visibility threshold, visible links, and possible links for quick viewing.
